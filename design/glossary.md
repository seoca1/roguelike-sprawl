# Game Glossary (게임 용어)

게임 내에서 사용하는 용어. 깁슨 원작의 용어와 구분하기 위해 별도 정의.

## 캐릭터 / 세계

| 용어 | 정의 | 비고 |
| --- | --- | --- |
| **Jockey / Cowboy** | 플레이어 캐릭터 직함 | (원작) console cowboy |
| **Handle** | 자키의 별명 / 핸들 | 플레이어 선택, 기본값 |
| **Grade** | 자키 등급 (1-up ~ 5-up) | 메타 진행 |
| **meatspace** | 사이버스페이스 밖 — *시각화되지 않음* | ADR-0009 |
| **wet** | 매트릭스 의식, "I'm in the wet" | (원작) |

## Story Archive (ADR-0009)

| 용어 | 정의 |
| --- | --- |
| **Story** | 전달된 뉴스 / 이야기 단위 |
| **Briefing** | 의뢰 briefing (런 시작 전) |
| **Result** | 의뢰 결과 (런 종료 후) |
| **World News** | 월드 이벤트 뉴스 (런 사이) |
| **Faction News** | faction 움직임 (T-A, Yakuza 등) |
| **Construct Memory** | construct와의 대화 기록 |
| **Archive** | 모든 story의 영구 저장 (메인 메뉴) |
| **Time Event** | 런 사이 시간 흐름 중 발생하는 자동 story |

## i18n (ADR-0010)

| 용어 | 정의 |
| --- | --- |
| **English (en)** | 1차 언어. 깁슨 원문 톤 직접 보존. |
| **Korean (ko)** | 보조 번역. 영어 텍스트의 한글 의역. |
| **Subtitle Mode** | 영어 위, 한글 아래 (자막) 표시. |
| **Replace Mode** | 한글만 표시 (영어 비표시). |
| **Off Mode** | 영어만 표시 (기본). |
| **Display Mode** | 설정의 번역 표시 옵션 (Off / Subtitle / Replace). |

### 고유명사 번역 원칙 (Proper Noun Retention)

고유명사(인명, faction, 모델, 브랜드)는 **영어 원문 그대로** 사용 가능. 일반 명사/서술/대사는 한국어 의역.

| 영어 (원문) | 권장 표기 |
| --- | --- |
| Case | Case (또는 케이스) |
| Molly Millions | Molly Millions (또는 몰리) |
| Wintermute / Neuromancer | 그대로 |
| Tessier-Ashpool | 그대로 |
| Ono-Sendai | 그대로 |
| ICE / construct / cyberspace | 영어 그대로 또는 한영 병기 |
| The Sprawl | The Sprawl (또는 스프롤) |
| Dixie Flatline | 그대로 |
| Freeside | 그대로 |
| Yakuza | Yakuza (또는 야쿠자) |
| SAMSARA / Maas / Sense/Net | 그대로 |

## 게임-특화 메카닉

| 용어 | 정의 |
| --- | --- |
| **HP** (Health / Integrity) | 데크의 무결성. 0 = flatline. |
| **Salvage** | ICE 격파 후 보상. HEAL(+20% max HP), FRAG, CRED, SKIP 중 선택 (ADR-0014). |
| **AP** (Action Points) | 메뉴 스킬 비용. 시간 경과로 회복. |
| **BW** (Bandwidth) | 동시 활성 프로그램 수. |
| **PW** (Processing Power) | 프로그램 복잡도 한계. |
| **HD** (Hidden Depth) | 탐지 회피 능력. |
| **Auto-Attack** | 양쪽 모두 일정 간격 자동 공격. |
| **Skill / Menu Skill** | 메뉴에서 선택하는 강한 공격/방어/유틸리티. |
| **Tier (T1~T5)** | 아이템/장비 등급. "레벨업" 표현. |
| **Edge** | 런 내 확률 보정 (Dice Pool 컨셉). |
| **Heat** | 의뢰 평판 / 위험 누적. |
| **Trace** | 위치 추적 진행 중. |
| **Alarm** | 시스템 경계 레벨. |

## 전투 (ADR-0003 — RT-MS)

| 용어 | 정의 |
| --- | --- |
| **RT-MS** (Real-Time with Menu Skills) | 실시간 + 메뉴 스킬 전투 시스템. |
| **Auto-Attack tick** | 자동 공격 한 번. 양쪽 동시. |
| **Menu Pause** | 스킬 선택 시 시간 일시 정지. |
| **Combat Flow** | 진입 → 실시간 자동 공격 → 메뉴 스킬 → 종료. |

## 난이도 (ADR-0012 — PPL & ZDR)

| 용어 | 정의 |
| --- | --- |
| **PPL** (Player Power Level) | 자키의 현재 힘 수치. 장비/프로그램/construct 종합. |
| **ZDR** (Zone Difficulty Rating) | Zone의 위험 수치. base + ICE + alarm + faction. |
| **Ratio** | PPL / ZDR. 1.0 = 균등. |
| **Status** | 비교 결과. SAFE / MATCH / TOUGH / DEADLY / FUTILE. |
| **SAFE** | Ratio > 1.5. 압도적 우위. green. |
| **MATCH** | 1.0 <= Ratio <= 1.5. 균등. cyan. |
| **TOUGH** | 0.75 <= Ratio < 1.0. 불리. yellow. |
| **DEADLY** | 0.5 <= Ratio < 0.75. 매우 위험. red. |
| **FUTILE** | Ratio < 0.5. 자살행위. dark red. |
| **Avoidance** | soft difficulty. 어려운 zone은 강제 진입 X. |
| **Phase / Stage** | 의뢰의 부분. 각 phase마다 ZDR 다름. |

| **Narrative Only** | Story Archive만 추가. |

## 디자인 원칙

이 용어들은 깁슨 톤을 따른다. 다른 사이버펑크 작품의 용어 (예: "netrunner", "aug")로 대체하지 않는다.

추가 용어는 시스템 명세에 정의될 때마다 본 문서에 추가.
