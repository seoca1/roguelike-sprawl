## 1. Source Works Overview

| Work | Year | Protagonist | Game Character | Timeline |
|------|------|-------------|---------------|----------|
| Neuromancer | 1984 | Case | Case (K) | Sets up T-A, Wintermute |
| Count Zero | 1986 | Bobby | Sil (Marly) | Parallel timeline, loa introduction |
| Mona Lisa Overdrive | 1988 | Kumiko | Kas (Kumiko) | Sequel timeline, declares war on T-A |

### 1.1 Timeline Relationships

```
Neuromancer (1984)
    │
    ├── Case jacks into matrix
    ├── Meets Wintermute/Neuromancer AI
    └── Merges with Wintermute

Count Zero (1986) — PARALLEL UNIVERSE
    │
    ├── Marly runs with Dixie Flatline crew
    ├── Loa gods appear in matrix
    └── Bobby becomes construct

Mona Lisa Overdrive (1988) — SEQUEL
    │
    ├── Kumiko declared "winner" of K Wachowski's bet
    ├── Sally Shearer construct
    └── The Wheel conspiracy

Game: ALL THREE TIMELINES MERGED
    │
    ├── Case works with Molly (post-Neuromancer)
    ├── Sil investigates loa (parallel to Case's missions)
    └── Kas confronts The Wheel (MLO timeline)
```

## 3. Matrix Architecture

### 3.1 cyberspace Structure

The matrix is the collective representation of global computer networks.

| Layer | Contents |
|-------|----------|
| Surface | Corporate entry points, data nodes |
| Mid-Zone | Industrial systems, ICE-heavy |
| Core | AI constructs, T-A mainframe |
| Hidden | Loa realms (Count Zero), Shadow zones |

| The Wheel | Yanaka/T-A | Industrial consciousness | Kas Ch1-Ch5 |
| Sally Shearer | Simstar industry | Unknown purpose | Kas Ch1-Ch3 |

### 3.3 Loa (Count Zero)

The unique element of Count Zero — voodoo gods that inhabit the matrix.

| Loa | Domain | Game Role |
|-----|--------|----------|
| Obeah | Death, rebirth | Present in Sil Ch2 |
| Papa Legba | Doorway, intersection | Present in Sil Ch2 |
| Shango | Lightning, justice | Present in Sil Ch3 |

#### Loa in Game

- **Sil's Specialty** — Only accessible to Sil
- **Voodoo Programs** — Programs powered by loa spirits
- **Opposition to T-A** — The loa oppose corporate AI

## 5. Timeline Conflicts & Mergers

### 5.1 Original Gibson Timeline

```
Neuromancer (1984)
    └── Case + Wintermute merge = Neuromancer AI

Count Zero (1986)
    ├── Bobby New Orleans (different character)
    └── Marly Krushkhova, 3Jane introduced

Mona Lisa Overdrive (1988)
    ├── Kumiko is "winner" of bet
    ├── Sally Shearer (different from Count Zero Sally)
    └── Different character relationships
```

### 5.2 Game's Merged Timeline

The game merges these into a single coherent world:

| Game Element | Source | Modification |
|--------------|--------|--------------|
| Case + Molly | Neuromancer | Extended post-book missions |
| Dixie Flatline | Neuromancer | Also works with Sil |
| 3Jane | Count Zero | T-A leadership in all arcs |
| Marly/Sil | Count Zero | Primary protagonist |
| Kumiko/Kas | Mona Lisa Overdrive | Extended pre-book missions |
| The Wheel | Original | Created from Yanaka-T-A merger |
| Loa | Count Zero | Expanded, tied to matrix core |

### 5.3 Key Differences from Source

| Source Element | Original | Game Version |
|----------------|---------|--------------|
| Case's胰腺 | Cured by 3Jane | Still damaged, seeking cure |
| Dixie Flatline | Dead by book end | Active construct throughout |
| Marly | Only in Count Zero | Exists across all arcs |
| 3Jane | Antagonist initially | Ally in later chapters |
| The Wheel | Doesn't exist | Kas's primary antagonist |
| Loa | Count Zero only | Accessible to all characters |

## 10. Creative Decisions

### 10.1 Merged Timeline Justification

The merged timeline allows:
- All three protagonists to interact across arcs
- T-A to be the unified antagonist
- The Wheel to connect Kas to the larger conspiracy
- Dixie Flatline to serve as guide for all characters

### 10.2 Original Elements

The following are **original game elements** not from source:

| Element | Justification |
|---------|---------------|
| The Wheel construct | Synthesizes MLO themes with original creation |
| Loa accessibility to all characters | Expands Count Zero's unique element |
| Case's persistent damage | Maintains dramatic tension |
| Extended mission structure | Provides gameplay loop |
| Grade/PPL system | Enables progression and difficulty scaling |

### 10.3 Non-Canonical Elements

These elements are **intentionally non-canonical** to Gibson's works:

| Element | Non-Canonical Because |
|---------|---------------------|
| Case cured | Neuromancer explicitly cures him |
| Dixie Flatline active | Neuromancer implies he fades after book |
| Marly meets Case | Different timelines |
| The Wheel as corporate AI | Original creation |
| Kumiko's father alive | MLO implies he's dead before story |

These non-canonical elements serve gameplay and narrative coherence while maintaining the spirit of Gibson's world.
