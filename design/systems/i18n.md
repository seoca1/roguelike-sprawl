## 목적

깁슨 원작의 **영어 톤 직접 보존** + 한국어 번역/자막 지원.
Display mode: **Off** (영어만) / **Subtitle** (영어+한글) / **Replace** (한글만).

## Translator API

```python
class Translator:
    lang: str
    _data: dict[str, Any]

    def t(key: str, **kwargs: Any) -> str:
        """Translate a dotted key with optional format arguments."""

    def has(key: str) -> bool:
        """Return True if the key exists AND is a string leaf."""

    def __call__(key: str, **kwargs: Any) -> str:
        """Shorthand for translator(key, **kwargs)."""
```

## Format Argument Handling (P2 #14)

`t(key, name="Case", count=3)` 호출 시:
- 정의된 kwargs → 정상 치환
- **누락된 kwargs** → `<name>` 마커 (예: `"Hello <name>!"`)
- 예외 절대 발생 안 함 (UI 텍스트 안전성)

이전: 누락 시 `KeyError` 발생 또는 raw `{name}` 노출.
이후: graceful fallback, 디버깅 시점에서 누락 명시.

## Translation Files

| 파일 | 언어 | 키 수 |
|---|---|---|
| `data/i18n/en.json` | English (primary) | ~200+ |
| `data/i18n/ko.json` | 한국어 (translation) | ~200+ |

깁슨 톤 (ICE, deck, program, construct, wetware, console cowboy 등)은
**영어 원문 그대로** 유지. 한국어에서는 의역.

## Display Modes (ADR-0010)

| 모드 | 동작 |
|---|---|
| **Off** | 영어만 표시 |
| **Subtitle** | 영어 (메인) + 한글 (작은 자막) |
| **Replace** | 한글만 표시 |

선택은 `state.settings.language_mode` 에 저장 (`engine/settings.py`).

## 데이터 흐름

```
i18n JSON files (en.json, ko.json)
    ↓ load
Translator(lang, data_dir)
    ↓ .t(key, **kwargs)
formatted string (with <name> markers for missing kwargs)
    ↓ render
Console.print(x, y, text, fg=...)
```

## Key Naming Convention

점(.) 으로 네임스페이스 분리:

```
matrix.entry_message      → "You jack in. The world goes gray."
matrix.flatline_message   → "You flatline. Static. Silence."
hub.title                → "HUB"
hub.credits              → "CREDITS: {amount}"
combat.victory           → "VICTORY"
combat.defeat            → "DEFEAT"
death.title              → "FLATLINE"
death.summary            → "REPORT"
```

깁슨 어휘 직접 보존 (예: "FLATLINE", "JACK OUT", "CONSTRUCT").
