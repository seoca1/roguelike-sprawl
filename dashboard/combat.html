📊 PPL & ZDR System

Player Power Level (PPL)

자키의 전투력을 정량화. 장비/프로그램/웨웨어 티어 합산.

PPL = (deck_tier × 3) + sum(programs) + (wetware_tier × 2) + (construct_tier) 예: Grade 1 (T1) loadout = 1×3 + 1 + 1×2 + 0 = 6 PPL

Zone Difficulty Rating (ZDR)

구역의 난이도. faction + zone 깊이 + 데이터 가치로 결정.

ZDR = base(zone) + modifier(faction) + risk(data_value) SAFE (0-2) · MATCH (3-5) · TOUGH (6-9) · DEADLY (10-14) · FUTILE (15+)

Status Calculation

if ZDR == 0: SAFE (회피 가능) if PPL > ZDR: MATCH (균형) if PPL >= ZDR×1.5: SAFE (쉬움) if PPL < ZDR: TOUGH/DEADLY (어려움)

🎬 VFX (시각 효과) — 5-Layer System

src/roguelike_sprawl/combat/effects.py — 모든 스킬·ICE·크리티컬에 ASCII 애니메이션, 파티클, 스크린 셰이크, 시네마틱 인트로/데스 시퀀스.

Layer 1 — 즉각 히트 피드백

  • 히트 플래시: 타겟 타일 120ms 백색/노랑
  • 플로팅 숫자: 데미지/힐, 위로 떠오르며 페이드 (800ms)
  • 히트 파티클: 6개 ASCII 문자 외측 비산 (중력 80)
  • 스크린 셰이크: 무거운 히트에 1.0-3.5 강도

Layer 2 — 15 스킬 애니메이션

  • ATTACK: [=> → 타겟 플래시
  • HEAVY_ATTACK: 차지 → 슬램 → 충격파
  • PIERCE: 관통 화살표 + 이중 플래시
  • MULTI_HIT: 3회 빠른 연타
  • DOT/POISON: 보라색 독성 파티클
  • SHIELD: ❖●❖ 육각 패턴
  • HEAL: 상승하는 + ✚ ❀
  • STUN: 회전하는 별 ✦·✦
  • LIFESTEAL: 빨간 선 + 자가 힐
  • COUNTER: 방패 후 반환
  • DETECT: 스캔 레티클
  • ... BUFF/DEBUFF/REGEN 포함 15종

Layer 3 — ICE 타입별 고유 연출

  • standard: [X_X] → · · ·
  • watchdog: WOOF! + 이빨 자국
  • goliath: 균열 + 무거운 임팩트 (red)
  • black: 글리치 ▓█▓▓█▓ + magenta
  • construct: 회전 기어 ⚙⚙⚙

Layer 4 — 상태이상 아이콘

  • P Poison · B Burn · S Stun
  • ❖ Shield · ↑ Buff · ↓ Debuff
  • + Regen · • DOT
  • 전투원 옆 표시, 버프/디버프는 테두리 색 변화

Layer 5 — 시네마틱 (Cinematic)

  • 전투 시작: 1.5-2초 ICE 이름 글리치 등장
  • 크리티컬: 250ms 슬로모션 + 강 셰이크 (3.5)
  • 데스 시퀀스: 5-8프레임 다단계 파괴
  • 콤보 카운터: 2x HIT! → 5x RAMPAGE!
