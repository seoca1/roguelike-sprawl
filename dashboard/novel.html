🎣 6 HookKind — 소설이 게임에 미치는 영향

NARRATIVE
단편의 첫 문단을 챕터 스크린에 표시. 12초 타이핑 효과.
→ chapter_view
EXCERPT
인라인 단편 발췌. 매트릭스 진입 시 또는 아이템 획득 시.
→ matrix / dungeon
EVENT
컷신/스토리 이벤트. 등장인물 등장, NPC 조우.
→ event_state
COMBAT
ICE 종류 시드 (STANDARD / BLACK). ice_kind 매개변수.
→ combat_state
ITEM
데이터 단편 / 인벤토리 추가. item_id 매개변수.
→ inventory
CINEMATIC
그래픽 노블 시네마틱. 자동재생.
→ graphic_novel

# (선택) 매핑 오버라이드

    "secondary": ["item"],
    "ice_kind": "BLACK"

# 게임 코드 수정 없이 끝

runtime.dispatcher.dispatch("new_story")
