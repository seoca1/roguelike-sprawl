🎮 Try It

Run the demo locally to see the graphic novel mode in action:

# Full prologue (12 scenes, 3 characters in random order) uv run python scripts/graphic_novel.py --mode prologue --seed 42 # Single character story (4 scenes) uv run python scripts/graphic_novel.py --mode novice uv run python scripts/graphic_novel.py --mode veteran uv run python scripts/graphic_novel.py --mode heretic # Korean translation uv run python scripts/graphic_novel.py --mode prologue --lang ko # Custom speed uv run python scripts/graphic_novel.py --mode novice --speed 5
