# Fonts

이 디렉토리는 게임에서 사용할 폰트 파일을 보관합니다.

## 기본 폰트

`terminal10x10_gs_tc.png` — libtcod 공식 10x10 터미널 폰트.

## 다운로드

```bash
make download-font
```

이 명령은 `scripts/download_font.py`를 실행하여 `terminal10x10_gs_tc.png`를 다운로드합니다.

원본: <https://github.com/libtcod/libtcod/blob/master/data/fonts/terminal10x10_gs_tc.png>
